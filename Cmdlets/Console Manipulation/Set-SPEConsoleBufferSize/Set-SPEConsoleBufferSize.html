Set-SPEConsoleBufferSize

Synopsis

Ändert den Speicher-Puffer der aktuellen PowerShell-Console

Syntax

Set-SPEConsoleBufferSize [-width] <Int32> [-height] <Int32> [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Ändert den Speicher-Puffer der aktuellen PowerShell-Console. Hierfür sind Höhe und Breite anzugeben

Parameters

-width <Int32>

Die Breite des neuen Puffers in Anzahl Zeichen

Position? 0

Accept wildcard characters? False

Default value

Accept wildcard characters? False
