Save-SPEXmlDocumentObjectAsUTF8

Synopsis

Speichert ein XmlDocument in eine XML-Datei mit UTF-8-Encoding

Syntax

Save-SPEXmlDocumentObjectAsUTF8 [-XmlDocumentObject] <XmlDocument> [-Path] <String> [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Speichert ein XmlDocument in eine XML-Datei mit UTF-8-Encoding. Das ist erforderlich, damit diese XML-Datei im späteren Verlauf sauber von weiteren Cmdlets ausgelesen werden kann, da grad deutsche Umlaute ansonsten in Steuerzeichen umgewandelt werden, die nicht mehr lesbar sind.
