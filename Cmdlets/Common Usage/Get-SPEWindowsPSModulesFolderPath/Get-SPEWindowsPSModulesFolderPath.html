Get-SPEWindowsPSModulesFolderPath

Synopsis

Erfasst den Standard-Windows-PowerShell-Ordner, in dem PS-Module abgelegt werden und gibt diesen als String aus.

Syntax

Get-SPEWindowsPSModulesFolderPath [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description
