Set-SPEGuidIncrement1stBlock

Synopsis

Erhöht die angegebene Guid im ersten Block von links um den Wert 1.

Syntax

Set-SPEGuidIncrement1stBlock [-guid] <Guid> [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Erhöht die angegebene Guid im ersten Block von links um den Wert 1.

Parameters

-guid <Guid>

Die Guid, deren erster Block von links um 1 erhöht werden soll.

Position? 0
Default value

Examples

PS C:\> $newGuid = Set-SPEGuidIncrement
$newGuid
$newGuid = Set-SPEGuidIncrement1stBlock -Guid $newGuid
