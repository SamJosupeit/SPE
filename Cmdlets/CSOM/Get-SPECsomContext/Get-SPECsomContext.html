Get-SPECsomContext

Synopsis

Erfasst den CSOM-Context einer SharePoint (Online) bezogenen URL in Bezug auf die angegebenen Credentials und gibt diesen aus.

Syntax

Get-SPECsomContext [-Url] <String> [[-Credentials] <PSCredential>] [-SPO] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Erfasst den CSOM-Context einer SharePoint (Online) bezogenen URL in Bezug auf die angegebenen Credentials. Da bei SharePoint Online die Credentials anders verarbeitet werden als bei einem OnPremise-SharePoint, ist für SharePoint-Online zusätzlich der Switch SPO zu setzen.

Examples

Beispiel 1 - SharePoint OnPremise

PS C:\> #Get the Credential-Object first:
$creds = Get-Credential 
#Get the Website-Context
$ctx = Get-SPECsomContext -Url "http://some.sharepoint.site" -Credentials $creds
