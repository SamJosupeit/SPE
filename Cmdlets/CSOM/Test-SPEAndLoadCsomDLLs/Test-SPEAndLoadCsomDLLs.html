Test-SPEAndLoadCsomDLLs

Synopsis

Stellt sicher, dass die DLLs Microsoft.SharePoint.Client und Microsoft.SharePoint.Client.Runtime geladen wurden.

Syntax

Test-SPEAndLoadCsomDLLs [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Stellt sicher, dass die DLLs Microsoft.SharePoint.Client und Microsoft.SharePoint.Client.Runtime geladen wurden. Dieses Cmdlet wird intern von den anderen CSOM-relevanten Cmdlets benutzt, um sicherzustellen, dass auch die entsprechenden Assemblies geladen wurden

Parameters

-InformationAction <ActionPreference>

Required? False

Default value

Accept wildcard characters? False

-InformationVariable <String>

Default value

Accept wildcard characters? False

ErrorAction, ErrorVariable, WarningAction, WarningVariable,
OutBuffer, PipelineVariable, and OutVariable. For more information, see
about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

Inputs

Outputs

Notes

Wird als Überprüfungsroutine in den SPECsom* Cmdlets eingesetzt.

Examples

Beispiel 1 - Allgemeine Benutzung

PS C:\> Function Some-CsomStuff
{
  [CmdletBinding()]
  param()
  Begin{ Test-SPEAndLoadCsomDLLs }
  Process{ [CODE] }
  End{}
}

Prüft am Anfang einer CSOM-bezogenen Function, ob die benötigten SharePoint-Client-DLLs geladen sind und lädt diese im Bedarfsfall nach.
