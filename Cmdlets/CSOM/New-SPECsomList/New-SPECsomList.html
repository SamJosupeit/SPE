New-SPECsomList

Synopsis

Erzeugt eine neue SharePoint-(Online-)Liste

Syntax

New-SPECsomList [-Web] <Web> [-ListTitle] <String> [-ListDescription] <String> [-ListTemplateId] <Int32> [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Examples

Beispiel 1 - CSOM OnPremise

PS C:\> #Get the Credential-Object first:
$creds = Get-Credential 
#Get the Website-Object
$web = Get-SPECsomWeb -Url "http://some.sharepoint.site" -Credentials $creds         $list = New-SPECsomList -Web $web -ListTitle "MyNewList" -ListDescription "This is my new list" -ListTemplateId 101

Erzeugt eine neue Liste vom Type "DocumentLibrary" in der Website "http://some.sharepoint.site"

Beispiel 2 - CSOM SharePoint Online

PS C:\> #Get the Credential-Object first:
$creds = Get-SPESPOnlineCredentials
#Get the Website-Object
$web = Get-SPECsomWeb -Url "http://some.sharepoint.site" -Credentials $creds -SPO       $list = New-SPECsomList -Web $web -ListTitle "MyNewList" -ListDescription "This is my new list" -ListTemplateId 101

Erzeugt eine neue Liste vom Type "DocumentLibrary" in der Website "http://some.sharepoint.site"
