Import-SPECsomDLLs

Synopsis

Import-SPECsomDLLs importiert die für das Client Site Object Model (CSOM) erforderlichen SharePoint-DLLs aus dem angegeben Ordner.

Syntax

Import-SPECsomDLLs [[-Path] <String>] [-UseDocumentManagement] [-UsePublishing] [-UseSearch] [-UseSearchApplications] [-UseServerRuntime] [-UseTaxonomy] [-UseUserProfiles] [-UseWorkflowServices] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Import-SPECsomDLLs importiert die für das Client Site Object Model (CSOM) erforderlichen SharePoint-DLLs aus dem angegeben Ordner oder, bei leerem PATH-Parameter aus dem Ordner "SharePointDLLS" im Module-Verzeichnis des SPE-Modules.
Es werden grundsätzlich die beiden DLLs "Microsoft.SharePoint.Client und Microsoft.SharePoint.Client.Runtime geladen, weitere können aber als Switch zusätzlich importiert werden.
