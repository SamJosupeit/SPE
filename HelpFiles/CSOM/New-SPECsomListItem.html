New-SPECsomListItem

Synopsis

Erstellt ein neues Item in einer SharePoint-(Online-)Liste.

Syntax

New-SPECsomListItem [-List] <List> [[-FieldValues] <ArrayList>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Erstellt ein neues Item in einer SharePoint-(Online-)Liste. Dazu müssen die Werte der einzelnen Felder als System.Web.UI.Pair-Werte in einer ArrayList abgelegt sein und übergeben werden.

Parameters

-List <List>

Das mit dem Cmdlet Get-SPECsomList erfasste List-Object

Required? True
Position? 0
Default value  
Accept pipeline input? false
Accept wildcard characters? False

-FieldValues <ArrayList>

ArrayList, die die Feldwerte als System.Pair-Objekte nach dem Schema First([FieldName])/Second([FieldValue]) enthält

Required? False
Position? 1
Default value  
Accept pipeline input? false
Accept wildcard characters? False

-InformationAction <ActionPreference>

Required? False
Position? named
Default value  
Accept pipeline input? false
Accept wildcard characters? False

-InformationVariable <String>

Required? False
Position? named
Default value  
Accept pipeline input? false
Accept wildcard characters? False

Outputs

Examples

Beispiel 1 - CSOM OnPremise

PS C:\> $creds = Get-Credential 
$web = Get-SPECsomWeb -Url "http://some.sharepoint.site" -Credentials $creds
$list = Get-SPECsomList -Web $web -ListTitle "MyList"
$Field1 = New-Object System.Web.UI.Pair("Title","MyNewItem")
$Field2 = New-Object System.Web.UI.Pair("AnotherField","ValueOfAnotherField")
$FieldValues = New-Object System.Collections.ArrayList
$FieldValues.Add($Field1)
$FieldValues.Add($Field2)
$newListItem = New-SPECsomListItem -List $list -FieldValues $FieldValues

Erzeugt ein neues List-Item in der Liste "MyList" innerhalb der Website "http://some.sharepoint.site" und gibt das erzeugte Item zurück. Field-Attribute müssen als System.Web.UI.Pair-Objects (Schema "Name" - "Wert") innerhalb einer System.Collections.ArrayList übergeben werden.

PS C:\> $creds = Get-SPESPOnlineCredentials
$web = Get-SPECsomWeb -Url "http://some.sharepoint.site" -Credentials $creds -SPO
$list = Get-SPECsomList -Web $web -ListTitle "MyList"
$Field1 = New-Object System.Web.UI.Pair("Title","MyNewItem")

$FieldValues = New-Object System.Collections.ArrayList
$FieldValues.Add($Field1)
$FieldValues.Add($Field2)
$newListItem = New-SPECsomListItem -List $list -FieldValues $FieldValues

Erzeugt ein neues List-Item in der Liste "MyList" innerhalb der Website "http://some.sharepoint.site" und gibt das erzeugte Item zurück. Field-Attribute müssen als System.Web.UI.Pair-Objects (Schema "Name" - "Wert") innerhalb einer System.Collections.ArrayList übergeben werden.
