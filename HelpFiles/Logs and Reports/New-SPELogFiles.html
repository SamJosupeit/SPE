New-SPELogFiles

Synopsis

Erzeugt neue Logfiles für PSE-Standard-Scripts

Syntax

New-SPELogFiles [-RecreateULS] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Erzeugt neue Logfiles für PSE-Standard-Scripts. Je nach Einstellung im Config-Script wird eine normale Text-Logdatei und/oder eine durch den ULS-Viewer lesbare ULS-Logdatei erzeugt. Dieses Cmdlet wird vom Cmdlet Get-SPEOrSetLogFiles benutzt, um neue Logdateien zu erzeugen.
Die Text-Logdateien werden nach dem Schema "Log_" + $ScriptName + "_" + $StringDateTime + ".log" benannt, die ULS-Logdateien nach dem Schema [ComputerName] + "-" + $StringDateTime + ".log", wie dies auch im SharePoint-Bereich der Fall ist. Das Namensschema der ULS-Dateien ist so erforderlich, um dem ULs-Viewer den sog. Runtime Feed zu ermöglichen, um so live die auftretenden Meldungen verfolgen zu können. Zur Ablage der Logdateien wird innerhalb des Script-Ordners durch das Cmdlet Get-SPEOrSetReportFiles ein Ordner "Log" erzeugt

Parameters

-RecreateULS <SwitchParameter>

Wenn gesetzt, wird das ULS-Logfile neu erstellt. Dies erfolgt automatisch, wenn die aktuelle ULS-Logdatei älter als die voreingestellte Zeit(maxAgeOfULSFile) oder größer als die voreingestellte Größe(maxSizeOfULSFile) ist.
