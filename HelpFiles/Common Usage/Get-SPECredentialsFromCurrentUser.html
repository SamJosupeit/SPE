Get-SPECredentialsFromCurrentUser

Synopsis

Öffnet ein Eingabefenster, in dem der aktuell angemeldete Benutzer seine Credentials eingeben kann. Diese können dann z.B. für den Zugriff auf eine SharePoint-Online-Website weiter genutzt werden.

Syntax

Get-SPECredentialsFromCurrentUser [-SPO] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Öffnet ein Eingabefenster, in dem der aktuell angemeldete Benutzer seine Credentials eingeben kann. Diese können dann z.B. für den Zugriff auf eine SharePoint-Online-Website weiter genutzt werden.

Parameters

-SPO <SwitchParameter>

Switch, um statt des standardmäßigen NETBIOS-Anmeldenamen das Feld für den Anmeldenamen frei zu lassen, um dort einen Anmeldenamen im Email-Format eingeben zu können.

Examples

Beispiel 1 - Standard

PS C:\> $creds = Get-SPECredentialsFromCurrentUser
$username = $creds.UserName
$securedPasswordString = $cred.Password

Öffnet ein PopUp zur Abfrage des Passworts mit Anzeige des Login-Namens (DOMAIN\USERNAME) des angemeldeten Benutzers
