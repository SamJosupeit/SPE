Set-SPEGuidIncrement

Synopsis

Erhöht die angegebene Guid um den Wert 1.

Syntax

Set-SPEGuidIncrement [[-guid] <Guid>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Erhöht die angegebene Guid um den Wert 1. Ist keine Guid angegeben oder ist diese NULL, wird eine neue Guid mit Nullen erzeugt und widergegeben. Da eine CorrelationID eine GUID ist, kann diese nicht einfach mit einem +1 inkrementiert werden, was dann durch dieses Cmdlet erfolgt.
