Get-SPEADUsersFromADContainer

Synopsis

Gibt ein oder alle Active Directory Benutzerkonten eines Active Directory Containers wieder.

Syntax

Get-SPEADUsersFromADContainer [-ADRoot] <DirectoryEntry> [[-sAMAccountName] <String>] [[-SearchScope] <String>] [[-Filter] <String>] [[-PageSize] <Int32>] [[-Properties] <String[]>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Get-SPEADUsersFromADContainer gibt ein oder alle Active Directory Benutzerkonten eines Active Directory Containers wieder.

Parameters

-ADRoot <DirectoryEntry>

Pfad zum Container der Active Directory Domain als LDAP-Link, z.B. "LDAP://OU=Users,DC=contoso,DC=com"

Required? True
Position? 0
Default value  
Accept pipeline input? false
Accept wildcard characters? False

-sAMAccountName <String>

Ist dieser Parameter gesetzt, wird der entsprechende Benutzer innerhalb des unter ADRoot angegebenen Containers gesucht und als Resultat ausgegeben. Ist dieser Parameter leer, werden alle Benutzer des Containers ausgegeben.

Required? False
Position? 1
Default value  
Accept pipeline input? false
Accept wildcard characters? False

-SearchScope <String>

Default-Wert ist "Subtree", kann aber gegen alle anderen Standard-Searchscopes getauscht werden

Required? False
Position? 2
Default value Subtree
Accept pipeline input? false
Accept wildcard characters? False

Default-Wert ist "objectClass=user", kann aber gegen alle anderen Standard-Filter getauscht werden

Required? False
Position? 3
Default value objectClass=user
Accept pipeline input? false
Accept wildcard characters? False

-PageSize <Int32>

Default-Wert ist 1000, kann aber individuell angepasst werden

Required? False

Examples

Beispiel 1 - Alle User in Container

PS C:\> $allUsersInUserContainer = Get-SPEADUsersFromADContainer -ADRoot "LDAP://OU=Users,DC=contoso,DC=com"

Gibt alle User im angegeben Container wieder

Beispiel 2 - User "Administrator" in Container

PS C:\> $GetUser = Get-SPEADUsersFromADContainer -ADRoot "LDAP://OU=Users,DC=contoso,DC=com" -sAMAccountName "Administrator"

Liest den Account des Users "Administrator" aus dem angegeben Container aus und gibt diesen wieder.
