Edit-SPETextFileByFilterToNewFile

Synopsis

Erfasst eine Text-Datei, überprüft den Text zeilenweise, ob diese bestimmte Filterbegriff enthalten und schreibt die Zeilen in eine neue Datei, wenn dem nicht so ist.

Syntax

Edit-SPETextFileByFilterToNewFile [-sourcePath] <String> [-targetPath] <String> [-filterStringArray] <Array> [-returnNewFile] <SwitchParameter> [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Erfasst eine Text-Datei, überprüft den Text zeilenweise, ob diese bestimmte Filterbegriff enthalten und schreibt die Zeilen in eine neue Datei, wenn dem nicht so ist. Mit diesem Cmdlet lassen sich zum Beispiel umfangreiche Log-Datei vor-filtern und von unnötigen Meldungen bereinigen.

Examples

Beispiel 1 - Allgemeine Benutzung

PS C:\> $src = "C:\QuellText.txt"
$trg = "C:\gefilterterText.txt"
$Filter = @("Peter","ist Berater")
$filteredText = Edit-SPETextFileByFilterToNewFile -sourcePath $src -targetPath $trg -filterStringArray $Filter -returnNewFile

Die Datei "C:\QuellText.txt" wird auf die Filter-Strings "Peter" und "ist Berater" geprüft, Zeilen, die diese Filter-Strings nicht enthalten, werden in die Datei "C:\gefilterterText.txt" geschrieben. Nach Abschluss wird die neue Datei als Object ausgegeben
