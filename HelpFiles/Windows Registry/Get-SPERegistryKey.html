Get-SPERegistryKey

Synopsis

Liest den Wert eines Registry-Keys mit dem angegebenen Namen unter dem angegebenen Pfad

Syntax

Get-SPERegistryKey [[-path] <String>] [[-key] <String>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Liest den Wert eines Registry-Keys mit dem angegebenen Namen unter dem angegebenen Pfad und gibt diesen als STRING aus

Parameters

-path <String>

Vollständiger Registry-Pfad zum auszulesenden Registry-Key

Outputs

Notes

Die Cmdlets Test-SPERegistryKey, Remove-SPERegistryKey, Set-SPERegistryKey, Get-SPERegistryKey, Restart-AndRun, Clear-SPERegistryAutostart sowie Set-SPERegistryScriptStatus und Get-SPERegistryScriptStatus werden von den SPE-Standard-Scripts genutzt, um Autostart-Schlüssel in die Windows Registry einzutragen, auszulesen, zu bearbeiten oder zu entfernen.

Examples

Beispiel 1 - Allgemeine Benutzung

PS C:\> Set-SPERegistryKey -path "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" -key "Restart-AndResume" -value "c:\ps1\autostart.ps1"
Get-SPERegistryKey -path "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" -key "Restart-AndResume"
c:\ps1\autostart.ps1

Get-SPERegistryKey liest den (z.B. mit Set-SPERegistryKey erstellten) Schlüssel "Restart-AndResume" aus dem Pfad "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" der Windows Registry aus und gibt dessen Wert als STRING aus.
