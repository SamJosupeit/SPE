Compare-SPESPFolderWithDirectoryAndDeleteMatchesInSPFolder

Synopsis

Vergleicht iterativ einen SPFOlder und dessen Subfolder mit einem FileSystem-Directory auf vorhandene Dateien. Dateien, die identisch im Directory existieren, werden im entsprechenden SPFolder gelöscht. SPFolder, die kein entsprechendes Gegenstück im Directory haben werden ebenfalls gelöscht.

Syntax

Compare-SPESPFolderWithDirectoryAndDeleteMatchesInSPFolder [[-sourceFolder] <SPFolder>] [[-directoryRoot] <String>] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Vergleicht iterativ einen SPFOlder und dessen Subfolder mit einem FileSystem-Directory auf vorhandene Dateien. Dateien, die identisch im Directory existieren, werden im entsprechenden SPFolder gelöscht. SPFolder, die kein entsprechendes Gegenstück im Directory haben werden ebenfalls gelöscht.
