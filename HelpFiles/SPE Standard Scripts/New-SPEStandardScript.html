New-SPEStandardScript

Synopsis

Erzeugt ein neues SPE-Standard-Script und speichert das im SPE-Standard-Verzeichnis ab. Gleichzeitig wird in der SPE-Config eine neue Sektion zum Script erstellt.

Syntax

New-SPEStandardScript [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]

Description

Erzeugt ein neues SPE-Standard-Script und speichert das im SPE-Standard-Verzeichnis ab. Gleichzeitig wird in der SPE-Config eine neue Sektion zum Script erstellt.
Vor der Erstellung des neuen Scripts werden verschieden Parameter abgefragt:
1. Name des neuen Scripts
2. Beschreibung des neuen Scripts (wird im einleitenden Dokumentations-Block hinterlegt
3. Soll das Script für einen Kunden erstellt werden? Wenn ja, wird der Kundenname abgefragt, der ebenfalls im Dokumentationsblock hinterlegt wird.
4. Soll das Default-Logging und -Reporting und der InfoHeader benutzt werden? Wenn Nein werden die einzelnen Möglichkeiten abgefragt.
5. Soll der TestModus benutzt werden?
6. Soll das Script mit Administrator-Rechten ausgeführt werden?
Daraufhin wird das Script erzeugt und die Parameter in die Config geschrieben.
